# Auto Backup Shell Script

This is a simple shell script that automates the process of backing up specified files or directories to a designated backup location.

## Features

- Automates the process of backing up files or directories.
- Allows you to specify the source and destination directories for the backup.
- Creates a timestamped backup folder to avoid overwriting previous backups.
- You can schedule the backup script to run automatically at regular intervals using cron jobs (Linux/macOS) or Task Scheduler (Windows).

## Prerequisites

- A Unix-based operating system (Linux/macOS)
- Shell (Bash) environment
